## **DEFINICIÓN**

a < b;  $a > b \rightarrow Designaldad\ estricta$  $a \le b$ ;  $a \ge b \rightarrow Designaldad\ no\ estricta$ 

## **PROPIEDADES**

1.- Transitividad

$$Si \ a < b \ y \ b < c \rightarrow a < c$$

2.-  $Si \ a < b$ 

$$a + c < b + c$$
$$a - c < b - c$$

3.-  $Si \ a < b \ y \ 0 < c$ 

$$a * c < b * c$$

$$\frac{a}{c} < \frac{b}{c}$$

4.-  $Si \ a < b \ y \ 0 > c$ 

$$a * c > b * c$$

$$\frac{a}{c} > \frac{b}{c}$$

5.- Sia < b, ambos de igual signo

$$\frac{1}{a} > \frac{1}{b}$$
