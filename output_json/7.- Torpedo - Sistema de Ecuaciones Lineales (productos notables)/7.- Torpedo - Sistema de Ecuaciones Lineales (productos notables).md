## **PRODUCTOS NOTABLES**

**Diferencia de cuadrados**

$$a^2 - b^2 = (a + b) * (a - b)$$

**Suma de cubos** 

$$a^3 + b^3 = (a + b) * (a^2 - ab + b^2)$$

**Diferencia de cubos**

$$a^3 - b^3 = (a - b) * (a^2 + ab + b^2)$$

**Cuadrado de binomio**

$$(a \pm b)^2 = a^2 \pm 2ab + b^2$$

**Cubo de binomio**

$$(a \pm b)^3 = a^3 \pm 3a^2b + 3ab^2 \pm b^3$$
